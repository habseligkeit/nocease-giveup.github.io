吾家有博初建成

发表于 2018-03-27 | 分类于 博客优化


  博客观江不语，使用hexo3+GitHub搭建，选择next-Pisces主题，参考了数篇教程，采用了很多已有的功能模块代码，比如您在页面右下角看到的小男孩haruto，使用的是hexo-helper-live2d模块；您点击鼠标时出现的五色烟火，使用的是fireworks模块；您在文章下的评论，使用了gitalk等等。至于哪些是我自己“设计”的，我翻了翻——背景图片是在必应壁纸上找的；将首页显示全文更改成了摘要显示，同时添加了阴影框。未来里，博客上会开始有我自己的设计，但目前，本博客是使用了各种前辈们已写好的代码。
  所以在此一定要感谢Gtihub这类开源平台

博客更新日志

2018/03/28

  1. 加入作者版权信息，这个直接将主题的配置文件_config.yml中post_copyright改为true就好。
  2. 修改超链接颜色为蓝色，文件next\source\css\_variables\base.styl，其他颜色修改也在该文件下。
  3. 在网站底部添加了访问统计量：不蒜子统计

遇到的主要问题

1.无法部署到Github

  我先后三次建博，前两次均应无法成功部署失败，主要表现在hexo s命令后，localhost:4000会显示我的更改，但hexo g -d却一直失败。这个问题有非常多的引发情况，我前两次建博用cmder输入命令，发现到后来很多更改无法部署，第三次建博，全部使用gitbash，这种情况再也没有出现过。

2.Gitalk Error
